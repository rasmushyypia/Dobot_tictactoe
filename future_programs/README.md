# Future Programs

Create all new projects here.

Recommended layout:

- `future_programs/<project_name>/README.md`
- `future_programs/<project_name>/requirements.txt` or `pyproject.toml`
- `future_programs/<project_name>/src/` or flat Python files if the project is small

Suggested use cases:

- vision-based board state detection
- local LLM move selection
- Dobot-safe command planning
- simulation or testing harnesses that do not touch the original app
