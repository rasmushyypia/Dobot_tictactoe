# ristinolla_ai

This workspace currently contains:

- `tictactoe/`: the original Dobot tic-tac-toe project copied from the Jetson Nano environment
- `python_bridge.py`: a separate screen-capture / local-LLM automation script
- `future_programs/`: the preferred location for all new development in this workspace

## Working Rule

The `tictactoe/` folder is treated as preserved reference code. New experiments and improvements should be built in separate folders under `future_programs/`.

## Planned Direction

The next stage for this repo is likely LLM-assisted play, perception, or control experiments built alongside the existing setup, not inside it.
